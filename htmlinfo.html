What is HTML?

Origin

Hypertext Markup Language (HTML) was origianlly conceived by physicist Tim Berners-Lee. He wanted to create an internet based hypertext. In late 1990s, Berners-Lee developed HTML specifications and wrote the browser and software server for it. Over the years, HTML has evolved into different versions, from HTML to HTML2, HTML3, HTML4, and eventually to the latest version of HTML5.

HTML is essentially a series of code and symbols used to markup texts for display in browsers. HTML can be used to implement structural elements, import media (images, videos), and user-interactive features such as forms. This allows the HTML coder to present information in a user-friendly, formatted, and attractive way to end-users who are requesting information from servers through the world wide web.

Main Elements

Two main elements of any HTML code is the head and body.

Head Meta data on document
Web page title
Chararacter encoding (i.e. utf-8)
Scripts
Styles
links to external sheets (i.e. css, ssi)
Body Visible context on browser
Media (photo/video)
Styled text
Structural elements (paragraphs, indents)
Interactive elements (forms)
Hyperlinks(urls, email, telephone)

In short, the head element contains information that is used by browers and search engines to extrapolate information regarding the html code. On the other hand, the body element contains information that is displayed to the end-user.

Block vs Inline-Block Elements

Both block and inline-block elements are the strcutural display of content. Block elements are treated as a new "block" starting on a new line and takes up the entire width. On the other hand, inline-block elements are also each treated as a separate "block" but they are aligned side by side with each other.
