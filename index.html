Internet vs World Wide Web

The internet is an umbrella term describing the global network. The global network consists of networks that are created through infrastructure & hardware (cables, cell towers, modems, etc) that allows communication and exchange between users. These indivdual networks are interconnected to form the internet. Each user connected to the internet is assigned an internet protocl (IP) address as their identification. The internet can be accessed through different methods, one of them being the world wide web.

World wide web (WWW) is part of the internet and is one way for users to access the internet. WWW uses HTTP protocol to transmit and receive data through the internet, which is displayed and viewed through browsers such as chrome and firefox. These browsers are designed to be able to read the http protocol and convert lines of code into visual information for end-users. WWW is part of the internet because it makes use of internet infrasctuture (servers, cables etc) and it uses the internet protocol (IP) to identify servers and users. As comparison, other alternatives to WWW to access the internet is the email (SMTP) or instant messageing (IM).
